# Flutter-Frenzy-G5
Flutter Frenzy Bootcamp by GDSC UM &amp; GDSC APU

### Checkout to *existing* branch
> git checkout [branch_name]

### Checkout to *new* branch
> git checkout [branch_name]

### Pull from remote>main into your local>your_branch
> git pull origin main

### Make a pull request
1. `git status` to view changed files
2. `git add [filename]`
3. `git commit --author "[name]" -m "[commit_message]"`
4. `git push origin [branch_name]`
